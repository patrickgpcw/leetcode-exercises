## Duplicate Integer

Given an integer array ```nums```, return ```true``` if any value appears <b>more than once</b> in the array, otherwise return ```false```.

####  Example 1:
```bash
Input: nums = [1, 2, 3, 3]

Output: true
```
#### Example 2:
```bash
Input: nums = [1, 2, 3, 4]

Output: false
```
